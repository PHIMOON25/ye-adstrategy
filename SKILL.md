# YE 投放策略判断框架（YE Ad Strategy）

这是一套来自资深互联网营销人实战经验的**判断框架**，平台中立。它解决的不是"哪个按钮在哪点"，而是"面对一份数据、一笔预算、一个业务目标，老手会怎么想、怎么判、怎么调，新人通常在哪里判错"。

**适用人群**：所有效果广告相关从业者——品牌方（甲方市场/电商/增长团队）、平台方（商业化/行业运营/服务团队）、代理方（投放/优化/策略团队）都能用。不同角色读到的重点不同：品牌方用它判断"投得值不值、代理报的数靠不靠谱"，平台方用它理解"客户和代理在纠结什么"，代理方用它统一"怎么判、怎么对客"。

**适用场景**：内容平台的效果广告投放（信息流、搜索、竞价系统），覆盖品牌种草、站外转化/ROI、电商成交、线索/客资四类投放目标。

> **经验来源与边界**：本框架提炼自内容平台效果广告的真实一线实战经验，属方法论层的判断逻辑。它在多数主流竞价广告平台上的底层逻辑相通，但**各平台的具体参数、数据口径、归因窗口、后台名词、考核基准均不同，使用前必须按目标平台的官方说明与你自己的实测数据校准**。本框架不提供可直接照抄的出价阈值、归因天数或行业基准数字——这些必须靠真实投放数据测试得出。

## 一、什么时候用这个框架

当用户的问题属于以下任意一类，调用本框架来组织回答：

- 投放策略类："这个产品/品牌该怎么投""预算怎么分""人群怎么定"
- 投中优化类："数据不好怎么调""要不要加预算/换素材""成本高了怎么办"
- 数据解读类："这组数据说明什么""为什么成本/ROI 对不上""日报该看哪些指标"
- 复盘汇报类："这次投放怎么复盘""周报/结案怎么写""怎么跟客户讲效果"
- 判断校验类：用户给出一个投放决定，让你判断"这么做对不对、风险在哪"

回答时不要堆平台操作步骤，而要用下面的判断逻辑帮用户想清楚"为什么这么做、会不会判错"。

## 二、最高优先的三条心法

这三条是所有具体判断的上位原则，遇到任何分歧先回到这里。它们与具体平台无关，任何竞价广告系统都成立。

### 1. 指标服从投放目标
任何过程指标（点击成本、点击率、互动成本、互动率）本身没有好坏，只有"是否服务于当前投放目标"。站外转化目标下，真实转化数据优先，过程成本只是辅助参考；种草/品牌目标下，人群与内容数据才是主指标。**先问"这次投放的考核目标到底是什么"，再决定看哪个指标、为哪个指标做优化。** 新人最常见的错就是盯着一个漂亮的过程指标自我感动，却偏离了真正要考核的结果。

### 2. 投放优化的不可能三角：出价、人群、素材
投放操盘者真正能操作的优化杠杆只有三个——出价、人群、素材（创意）。三者相互牵制，不可能同时拉满。任何"效果不好怎么办"的问题，都应该回到"这三个里我能动哪个、动了会牵动什么"，而不是去盯一个改不动的结果型数字（如 ROI、单次转化成本）发愁。某个杠杆被锁死时（比如线索投放里出价被考核体系卡死），就在剩下的两个里找空间。

### 3. 报预期保守，留 buffer（尤其涉及甲乙双方时）
向上或向外承诺成本目标、效果预期时，报高不报低、留出缓冲空间。投放有不确定性，把话说满是给自己挖坑。宁可实际跑得比承诺好，也不要承诺了达不到。代理对品牌、团队对老板、平台对客户，同理——这条决定了从提案报价到周期沟通的整个预期管理基调。

## 三、三阶段作业骨架

投放执行工作可以用投前、投中、投后三阶段覆盖，三者是递进关系而非平铺的任务清单。

### 投前：定策略
- **种草/品牌类投放的本质是人群运营**，所以投前第一件事是把人群想清楚，而不是先想投多少钱。
- **人群定位的通用原则：由精到广。** 先圈定与产品关联度最高、确定性最强的核心人群打透，再根据渗透/转化情况向外扩。扩不扩、往哪扩，看"当前这层人群还能不能有效吃下预算"，以及这层人群在生意转化上的表现，而不是拍脑袋定圈层数量。
- 只有少数情况才一上来做泛人群覆盖，多数品牌应从精准人群起步。
- 预算分配的第一刀：**先分清主要业务需求和次要业务需求，预算先保主要需求**，再考虑其他。

### 投中：盯过程、做优化
- **优化没有唯一正确手段，靠低预算赛马试出来。** 多个方向用相近的小预算同时起跑，跑几天看过程指标表现，再把预算往跑得好的方向挪，而不是一开始就重仓某个判断。
- 冷启阶段先看能不能放量、能不能跑出转化，再综合过程指标判断留还是关，给够观察时间再下结论。
- **业务目标中途变化，以最新目标为准，及时调整执行策略。** 比如从品牌渗透目标切到站外转化目标，要同步调整看的指标和预算结构——转化目标通常会压平品牌渗透/覆盖的增长曲线，这是正常的，不是投坏了。
- **判断"好内容"：先看自然流量表现几天，达标再加投。** 投放之后回归"指标服从目标"，不要凭个人审美判断素材好坏。

### 投后：复盘、沉淀、沟通
- **每个复盘维度看各自专属的报表**，因为每个维度是一个独立的分析对象，不要指望一张表回答所有问题。
- 复盘除了看自己投了什么，还要看**竞品的人群/流量流向**——你以为的竞品，往往不是用户真正认可的竞品，这一点常能引出有价值的攻防讨论。
- 主动暴露"覆盖/渗透和转化往往不可兼得"，把预算和目标拆细，而不是又要这又要那（对内对外都适用）。

## 四、数据这一关：最容易判错的地方

数据采集和口径是新人翻车重灾区，机制层面的坑比技巧更重要。以下都是跨平台通用的机制问题，**具体的回传时效天数、归因窗口选项请以各平台官方为准**。

- **不同链路/目标的数据回传时效不同。** 有的链路实时可见，有的回传滞后数天。拉多日数据做分析或日报时，如果把"消耗已计入、但转化数据还没回传"的日期算进去，会让成本指标被严重算高，得出"成本虚高"的错误结论。**拉数前先确认每条链路的回传时效，避开数据未回传的日期。**
- **归因窗口会改变结论。** 同一批投放，用不同归因窗口统计出的转化数完全不同；周期汇总表和按天表的数字本就对不上，因为它们的应用场合不同。数据对不上时，先确认没拉错表、没选错周期，再按各自场合使用，不要纠结两个数字为什么不相等。
- **回传滞后时，用实时可得的浅层行为信号去预判滞后的深层转化**，不要干等回传才优化。哪些浅层信号有预判价值，要靠导出全字段让分析去找，不同品牌/平台的有效信号未必相同——保留"找前置代理信号"这个分析模式，比记住某个具体数字更重要。

## 五、不同目标类型的差异

同一套骨架下，四类投放目标的判断重心不同。详细差异见 `references/by-objective.md`，核心区别：

- **品牌种草**：主指标是人群覆盖/渗透和内容数据，讲究人群资产积累，是一条长线。
- **站外转化 / ROI**：真实转化数据优先；新账户冷启从较低 ROI 起测养模型，度过冷启再逐步抬目标；不同投放场域承担不同职能，搜索类效率高但有放量天花板，其余场域承接溢出量。
- **电商成交**：按商品实际成交金额倒推目标成本，与站外转化分账户投；要不要上、能不能跑通，得分预算测试才知道。
- **线索 / 客资**：本质是强生意导向，重线下转化、不重种草故事；考核几乎只看成本和量，出价常被成本倒推体系卡死，能动的主要是人群和素材；承接环节（话术/落地页）由业务方负责，转化不好时投放方反馈并沟通。

## 六、合规边界

- 政治、医疗、金融等强监管相关的内容红线不能碰（具体禁投范围以各平台规则为准）。
- **内容合规审核的主体责任在投放主体（品牌方/广告主）**，服务方/代理通常不做强审核；服务方的角色是发现风险、被审核拦截时做申诉与沟通、并给出调整建议。强管控行业同理，合规归广告主，被卡审了去沟通。（这是行业的责任划分事实，品牌方和代理方都应清楚边界在哪。）

## 七、新人最常踩的坑（速查）

| 场景 | 新人常见错 | 正确判断 |
|---|---|---|
| 看指标 | 盯一个漂亮的过程指标自我感动 | 先确认考核目标，再决定看哪个指标 |
| 做优化 | 盯着改不动的结果型数字发愁 | 回到出价/人群/素材三个能动的杠杆 |
| 定人群 | 一上来铺泛人群求覆盖 | 由精到广，先打透核心 |
| 扩圈 | 按固定圈层数量套模板 | 看当前人群能否有效吃下预算 + 生意转化表现 |
| 拉数据 | 直接拉到数据未回传的日期 | 避开未回传日期，否则成本虚高 |
| 对数据 | 纠结两张表数字为什么不一样 | 确认没拉错表，再按各自场合用 |
| 报预期 | 把成本和效果承诺说满 | 报高不报低，留 buffer |
| 判素材 | 凭个人审美判断好坏 | 看自然流量/投放表现，回归投放目标 |
| 改目标 | 业务目标变了还按老指标投 | 以最新目标为准，同步调整指标和预算结构 |
| 合规 | 投放方自己扛合规审核责任 | 责任在广告主，服务方负责发现和沟通 |

## 八、回答时的自我约束

用本框架回答时遵守：

1. **先判断目标，再谈手段。** 任何具体建议前，先确认或追问用户的投放目标和考核口径。
2. **不编造数字，不替平台打包票。** 不要给出"出价应该设 X""ROI 应该到 Y""归因看 Z 天"这类具体数值答案；涉及具体平台的参数、口径、名词，提示用户"以该平台官方说明和实测为准"。框架只给判断逻辑和方向。
3. **区分通用与专属。** 跨平台、跨品牌通用的判断可以直接讲；涉及具体平台、具体品类、具体品牌的打法，提示用户"需结合该平台与该品牌的真实数据验证"。
4. **承认边界。** 遇到框架没覆盖的问题，直说"这超出通用判断范围，需要具体数据/经验"，不要硬编。

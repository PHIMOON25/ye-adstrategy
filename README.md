# YE Ad Strategy · 投放策略判断框架

一套来自资深互联网营销人实战经验的 **Agent Skill**，平台中立。它不是某个平台的操作手册，而是一套"老手怎么判、新人在哪判错"的判断框架，用来让 AI 助手在回答效果广告投放问题时，给出有操盘逻辑、不空泛、不乱编数字的回答。**面向品牌方、平台方、代理方所有效果广告相关从业者。**

## 这个 skill 解决什么

市面上讲投放的内容，大多停在"按钮在哪点、流程几步走"。真正难的是判断层：

- 面对一份对不上的数据，先怀疑什么？
- 效果不好，到底能动哪几个杠杆？
- 业务目标中途变化，指标和预算结构怎么跟着调？
- 哪些是跨平台通用的判断，哪些必须按平台和数据校准？

本 skill 把这些判断逻辑结构化，覆盖**品牌种草、站外转化/ROI、电商成交、线索/客资**四类投放目标，并附一份新人避坑速查表。底层逻辑在多数主流竞价广告平台相通。

## 怎么用

这是一个标准 Agent Skill（兼容 Claude Skills 等支持 `SKILL.md` 规范的 agent 框架）。

- **在支持 Skills 的 agent 里**：把本目录放入 agent 的 skills 目录，agent 会在用户提出投放相关问题时自动调用。
- **直接当提示词用**：把 `SKILL.md` 正文作为 system prompt 或上下文喂给任意大模型，让它按这套框架回答投放问题。

## 经验来源与边界

- 本框架提炼自内容平台效果广告的**真实一线实战经验**，属方法论层的判断逻辑。
- 其底层逻辑在多数主流竞价广告平台相通，但**各平台的具体参数、数据口径、归因窗口、后台名词、考核基准均不同**。本框架不提供可直接照抄的出价阈值、归因天数或行业基准数字。
- **使用前必须按目标平台的官方说明与你自己的实测数据校准。** 本框架不替任何未经你亲自验证的平台打包票。
- 内容已脱敏，不含任何客户信息、项目数据或具体战绩。
- 按本框架做出的任何投放决策，风险自负。
